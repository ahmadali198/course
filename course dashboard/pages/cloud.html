Cloud Computing Fundamentals

Instructor: Michael Harris

Get started with cloud computing in this foundational course. Learn about different cloud service models (IaaS, PaaS, SaaS), cloud providers like AWS, Azure, and Google Cloud, and how to manage and deploy cloud-based applications.

  • Price: $299

Curriculum

  • Introduction to Cloud Computing
  • Cloud Service Models and Deployment
  • Working with AWS
  • Managing Azure Resources
  • Google Cloud Platform Essentials
  • Cloud Security and Best Practices
  • Project: Deploying a Cloud Application
